sysprep
=======

my personal bootstrap scripts


# Bootstrap Fedora Workstation 25+

    $ ./bootstrap.fedora.sh
    $ ./bootstrap.git.preference.sh


# Bootstrap macOS Sierra 10.12.0+

    $ ./bootstrap.mac.sh
    $ ./bootstrap.git.preference.sh

# Bootstrap raspbian 9+ (raspberry pi 3)

    $ ./bootstrap.rspi3.sh
    $ ./bootstrap.git.preference.sh
