sysprep
=======

my personal bootstrap scripts


# Bootstrap Fedora 25+

    $ ./bootstrap.fedora.sh
    $ ./bootstrap.fedora.repository.sh
    $ ./bootstrap.git.preference.sh


# Bootstrap macOS Sierra

    $ ./bootstrap.mac.sh
    $ ./bootstrap.git.preference.sh
